## 常用方法

### chunk

`chunk(array, [size=1])`

将数组以 size 为单位分割成几份，返回二维数组

#### 使用场景

- 如调用后端批量校验接口时，后端一个接口承受不了巨量数据的校验，此时前端可以分割被校验的数据，拆分成多个 http 请求，使用 `Promise.all` 并发请求数据

- 深拷贝

### isEqual

比较两个对象的内容是否相同

#### 使用场景

- 用户点击 **退出** 时，二次确认是否退出，比较数据备份与当前数据的变化
